# State Counter Analytics - Frontend

React + Vite frontend for State Counter Analytics.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on http://localhost:3000

## Features

- Project management
- Real-time dashboard
- Visitor tracking
- Traffic source analysis
- Page analytics
- Visitor journey visualization
- Export reports

## Tech Stack

- React 18
- React Router
- Vite
- Axios
- Lucide Icons
